## 📊 Key Projects

### 🔹 Executive Sales Performance Dashboard
**Tools:** Power BI · SQL · Excel  
Built an end-to-end sales analytics dashboard consolidating data from 5 regional sources. Delivered automated weekly reporting for senior leadership, reducing manual reporting effort by **70%** and surfacing a pricing anomaly that recovered **$200K+ in annual revenue**.

---

### 🔹 Customer Segmentation & Retention Analysis
**Tools:** Python (Pandas, Scikit-learn) · SQL · Power BI  
Performed RFM (Recency, Frequency, Monetary) segmentation across 50,000+ customer records. Identified a high-value churn-risk cohort, enabling targeted retention campaigns that improved 90-day retention by **18%**.

---

### 🔹 KPI Framework & Operational Analytics
**Tools:** SQL · Power BI · DAX  
Designed a company-wide KPI tracking system, defining 30+ metrics aligned to business objectives. Created live Power BI dashboards consumed by 4 departments, replacing fragmented spreadsheet reporting with a single source of truth.

---

### 🔹 Marketing Funnel & Conversion Analysis
**Tools:** Python · SQL · Excel  
Analyzed a 12-month marketing funnel across 6 acquisition channels. Identified a conversion drop-off at the consideration stage using cohort analysis, informing a content strategy shift that improved MQL-to-SQL conversion by **22%**.
