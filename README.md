## 〔 01 〕 About Me

<img align="right" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" width="320" alt="Data Analytics Animation"/>

I'm **Mohamed Desouky**, a professional **Data Analyst** based in Egypt 🇪🇬 with a focused expertise in transforming complex, multi-source datasets into clear, actionable intelligence.

My core philosophy: **data without context is noise — my job is to create the signal.**

- 🔬 &nbsp; I design **end-to-end analytics pipelines** — from raw ingestion to polished executive dashboards
- 📊 &nbsp; I specialize in **financial & business data** — sales performance, market behavior, operational KPIs
- 🤖 &nbsp; I build tools that layer **AI-assisted analysis** on top of structured data for smarter decision support
- 🌐 &nbsp; I believe dashboards should be **beautiful and fast** — not one or the other
- 🎯 &nbsp; Currently engineering **NEXUS** — a real-time Forex data analytics showcase platform

> **NEXUS** is not a trading application — it is a **data analytics engineering showcase** built on live financial market data. It demonstrates the ability to architect full-stack data solutions: from ingestion pipelines and algorithmic indicator computation, to AI-powered text analysis and interactive real-time visualization.

### 🧠 What NEXUS Demonstrates as a Data Analyst

| Analytical Skill | How NEXUS Applies It |
|---|---|
| 📥 **Real-Time Data Ingestion** | Live price feed pipelines via APIs; streaming tick data processing |
| 🔢 **Quantitative Computation** | From-scratch implementation of RSI, MACD, Bollinger Bands, EMA using Pandas & NumPy |
| 📰 **NLP & Sentiment Analysis** | Financial news aggregation → VADER/TextBlob scoring → directional bias signal |
| 🔀 **Multi-Source Data Fusion** | Merging structured price data with unstructured news text into a unified confluence score |
| 📊 **Advanced Visualization** | TradingView embedded charts + custom Plotly KPI gauges + responsive dark-mode Streamlit UI |
| 🏗️ **Pipeline Architecture** | Modular codebase: data layer → indicator engine → AI layer → UI components |
| 📱 **Deployment Engineering** | PWA-optimized, deployed on Streamlit Community Cloud with CI-ready structure |

<img align="left" src="https://media.giphy.com/media/3oKIPEqDGUULpEU0aQ/giphy.gif" width="190" alt="E-Commerce Dashboard GIF"/>

**Domain:** E-Commerce Analytics &nbsp;|&nbsp; **Tool:** Power BI / Streamlit &nbsp;|&nbsp; **Data:** Salla Platform

A comprehensive business intelligence dashboard built for an active e-commerce store on the **Salla platform** — the leading Arab e-commerce SaaS. Covers revenue trends, product performance segmentation, customer behavior cohorts, and fulfillment efficiency. Designed for non-technical business owners to make fast, confident decisions.

**Analytical Highlights:**
- Revenue attribution by product category with seasonal trend decomposition
- Customer segmentation using RFM (Recency, Frequency, Monetary) scoring
